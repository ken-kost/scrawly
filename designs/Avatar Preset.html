Drawing rules

Every icon is composed from circles, simple curves and rounded strokes — no fussy detail. Same 3.2px ink line throughout, so the set feels like one hand drew it.

Color logic

Six muted tones rotate across the grid so no two neighbours share a background. The user picks any tone independently from the icon, giving 30 × 6 = 180 combos.
