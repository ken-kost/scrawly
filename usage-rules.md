# Rules for working with Hologram

## Understanding Hologram

Hologram is a full-stack Elixir web framework that compiles Elixir to JavaScript and runs code in the browser. It provides a declarative component-based architecture with client-side state, automatic code distribution, and HTTP/2 client-server communication. Read documentation *before* attempting to use its features. Do not assume patterns from Phoenix LiveView or other frameworks apply — Hologram has its own conventions.
